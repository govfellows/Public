# Public

Public-facing repository for sharing finished artifacts and innovations from the Innovation Fellowship.

## What's Here

This is where we publish polished, final versions of our work for public visibility. Drafts and working documents belong in the other project spaces.

## How to Contribute

- **Upload a file:** Click **Add file** → **Upload files** above, drag your file in, then click **Commit changes**
- **Edit a file:** Click on the file, then click the pencil icon to edit

> **Note:** This repo is publicly visible — anyone on the internet can see its contents. Only upload materials that are ready for public sharing.
